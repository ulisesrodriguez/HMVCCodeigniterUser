Descriptión

   This module allow create, update, delete and view profiles of user registered in your sistem.
   Language Spanish and English

Install


  1.- Download. https://github.com/ulisesrodriguez/user.git"

  2.- Download Codeigniter. http://ellislab.com/codeigniter

  3.- Download extencion HMVC for codeigniter. https://github.com/Crypt/Codeigniter-HMVC

  4.- Uncompress the file rar of codeigniter and copy to C:\\yourserver\www.  

  5.- Uncompress the file rar of extención HMVC and configure.  

  6.- Uncompress the file rar and copy the module in this folder application/modules/.    

  7.- Run the file application/modules/user/assets/sql/sql.sql  in PHPMyAdmin

  8.- GO URL http://youcodeigniterinstalation/user/view_list.

Config ajax javascript


  Open the file application/modules/user/assets/script/users.js
  set the var url with the url absolute the your codeigniter installation
  Example: http://localhost/codeigniter/
  var url = '';

Config pagination


  Open the file  application/modules/user/controllers/user.php

  In Method

  public function view_list( $begin = 0 )

  $config['per_page'] = 20;  //This set the number of record for page

Config language


  Open the file application/modules/user/controllers/user.php
  Change the value of vars, for default the value is english  if you want change to, set this to spanish


  // Languege

    public $language = 'english';

  // Language Validator

    public $lang_validator = 'english';
